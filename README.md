# Pizza Div

A math game that teaches kids long division through pizza sharing. No math symbols — just play.

**Live at: [pizzadiv.amiheines.com](https://pizzadiv.amiheines.com)**

## How to Play

- **Swipe down** on pizza → distribute pieces to hungry mice (one per mouse)
- **Swipe right** from the knife → cut all whole pizzas into 10 slices each
- Feed all the mice to complete the level!

## Levels

1. 1 pizza, 1 mouse — learn to distribute
2. 1 pizza, 10 mice — learn to cut, then distribute
3. 2 pizzas, 5 mice — multiple pizzas
4. 1 pizza, 5 mice — leftovers
5. 3 pizzas, 10 mice — remainders

## Stack

Vanilla HTML/CSS/JS — no build step, no framework. PWA with offline support.

## Deploy

```bash
./deploy.sh
```
